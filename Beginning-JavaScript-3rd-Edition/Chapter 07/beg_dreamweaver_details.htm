Book overview

With this book you'll quickly be creating powerful, dynamic web sites with Dreamweaver MX 2004 - Macromedia's powerful, integrated web development and editing tool. You'll learn how to use the tools and features of Dreamweaver to construct three complete sites using HTML, JavaScript, Active Server Pages (ASP), and databases.
